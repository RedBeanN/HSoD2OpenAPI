# API-gacha

## 十连扭蛋 [https://api.redbean.tech/gacha/:pool](https://api.redbean.tech/gacha/high)

### 参数

#### `pool: String`

对应不同卡池

| pool | 对应蛋池 |
| ---- | ---- |
| high | 公主 |
| custom | 魔女 |
| special | 魔法少女 |
| middle | 大小姐 |

### 返回值

`Array[Object]` 一个包含 10 个对象的数组.

对象的键值:

| key | value | type |
| --- | ----- | ---- |
| title | 装备名 | String |
| isGod | 是否为神器 | Boolean |
